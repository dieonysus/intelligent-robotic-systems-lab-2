# Lab 2
## Task

The robot is asked to find a light source and go towards it, while avoiding collisions with
other objects, such as walls, boxes and other robots. The robot should reach the target as
fast as possible and, once reached it, it should stay close to it (either standing or moving).
For physical constraints the wheel velocity cannot exceed the value 15 (i.e., 15−2 m/s).
The robot (a footbot) is equipped with both light and proximity sensors.
